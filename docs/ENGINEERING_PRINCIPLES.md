# Sentinel Engineering Principles

Version: v0.9.5

## 1. Reliable Intelligence begins with reliable data.

Every strategic assessment is only as trustworthy as the quality, consistency and actionability of the underlying data.

## 2. Reliable engineering begins with measurable results.

Important engineering decisions should be backed by repeatable measurements whenever possible.

## 3. Never optimize assumptions. Benchmark them.

When multiple implementations may solve the same problem, Sentinel should compare them using representative project data.

## 4. OCR is only a sensor.

Sentinel must not become dependent on one OCR engine. OCR providers may change; the intelligence pipeline remains stable.

## 5. The better idea wins.

Architecture and implementation decisions are judged by project value, not ownership.
