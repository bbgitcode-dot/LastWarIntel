# Glossary

Version: v0.9.0-pre1

## Purpose

This glossary defines the canonical domain language used throughout Sentinel.

## Observation
Raw imported information from any source.

## Snapshot
Immutable representation of the game state at a specific point in time.

## Difference
Normalized change between two snapshots.

## Fact
Objective statement derived from one or more differences.

## Evidence
Collection of related facts supporting a hypothesis.

## Indicator
Strategic characteristic derived from evidence.

## Reasoning
Deterministic interpretation of indicators producing hypotheses.

## Assessment
Description of a strategic situation. Assessments explain *what is happening*.

## Value
Prioritization of an Assessment for a specific mission.

## Recommendation
Suggested human action derived from Assessment and Value.

## Confidence
Measure of evidence strength, not probability.
