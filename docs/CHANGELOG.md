# [0.8.0] - 2026-06-28

## 🚀 Recruitment Intelligence MVP

### Added

* Recruitment Intelligence module
* Recruitment Context model
* Recruitment Context Builder
* Recruitment Value Calculator
* Recruitment Value Facade
* Recruitment Value integration into Sentinel
* Recruitment-aware WatchTargets
* Recruitment scoring pipeline
* Decision Snapshot improvements

### Recruitment Metrics

Implemented recruitment evaluation based on:

* Talent Value
* Structural Health
* Recruitability
* Whale Density
* Recruitable Density
* Elite Density
* Recruitment Momentum (foundation)

### Intelligence

Added support for:

* Recruitment Context generation
* Strategic Recruitment Value
* Explainable recruitment scoring
* Modular recruitment architecture

### Architecture

* Completed end-to-end Recruitment Intelligence pipeline
* Introduced layered Recruitment Context generation
* Extended WatchTarget with RecruitmentValue
* Preserved separation between Assessments and Values
* Improved modular Intelligence architecture
* Unified Intelligence → Operations workflow

### Operations

Recruitment Intelligence now flows automatically through

* Watchlist
* Reports
* Morning Briefing
* Sentinel Orchestrator

### Documentation

Complete documentation refresh.

New documentation includes:

* README
* ARCHITECTURE
* SENTINEL
* INTELLIGENCE
* VISION
* ROADMAP

# Philosophy

Sentinel does not measure success by the number of implemented features.

Success is measured by one simple question:

> **Does this release help alliance leadership make better strategic decisions?**

Every version should move Sentinel one step closer to becoming an explainable strategic intelligence platform.
