# What is Sentinel?

Sentinel is a strategic intelligence platform for **Last War: Survival**.

Unlike traditional dashboards, Sentinel does not focus on displaying statistics.

Its purpose is to transform thousands of individual observations into clear, explainable strategic recommendations that help alliance leadership make better decisions.

Instead of asking:

> *"What happened?"*

Sentinel answers:

> *"What matters?"*

and

> *"What should we do next?"*

---

# Vision

Large alliances generate enormous amounts of information.

* Daily snapshots
* Alliance rankings
* Player transfers
* Power changes
* Whale movements
* Recruitment opportunities
* Structural instability

Most leadership teams try to process this manually using screenshots, spreadsheets and experience.

Sentinel automates this process.

It continuously transforms raw observations into intelligence, allowing commanders to focus on strategic decisions instead of collecting data.

---

# Core Principles

Sentinel follows five fundamental principles.

## 1. Intelligence over Information

Displaying data is easy.

Understanding what actually matters is difficult.

Sentinel prioritizes relevant intelligence instead of overwhelming users with numbers.

---

## 2. Explainability

Every recommendation must be explainable.

Users should always be able to answer:

* Why is this alliance interesting?
* Why is this server unstable?
* Why was this recommendation generated?

Every decision can be traced back through the complete reasoning chain.

---

## 3. Human First

Sentinel supports decision makers.

It never replaces them.

The final decision always belongs to alliance leadership.

---

## 4. Deterministic Intelligence

Sentinel is not a black box.

Given the same observations, Sentinel always produces the same conclusions.

This makes every recommendation reproducible and verifiable.

---

## 5. Strategic Focus

Metrics never exist for their own sake.

Every metric must support a strategic decision.

If a metric cannot answer an important strategic question, it should not exist.

# Explainable Intelligence

One of Sentinel's primary design goals is explainability.

Every recommendation follows the same reasoning chain.

```text
Observation

↓

Fact

↓

Evidence

↓

Indicator

↓

Reasoning

↓

Assessment

↓

Value

↓

Recommendation
```

Nothing is hidden.

Every recommendation can be traced back to the original observations.
