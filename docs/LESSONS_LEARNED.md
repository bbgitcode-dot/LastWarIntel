## v0.9.5.93 Lesson – Review Is Not Operational Truth

`PENDING REVIEW` placeholders are useful internal evidence, but they must not appear as normal operational ranks. If a quarantined row becomes THP rank 102 or 103, Sentinel has mixed review state into Operational Truth. v0.9.5.93 separates these surfaces: normal sheets show accepted rows; review sheets show unresolved rows.

## v0.9.5.93 Lesson – OCR Match Is Not Identity Fidelity

`Joncollins21` and `Joncollinszl` are similar for OCR, but they are not the same operational identity. The same applies to alliance tags such as `DAY` and `daY`. Sentinel must preserve exact observed identity for historical tracking and transfer intelligence.

## v0.9.5.92 Lesson – Over-Quarantine Can Also Harm Truth

v0.9.5.91 protected recall but exported many correct Server 551 rows with `rank=None`. The lesson is that Data Guard must not only block false truth; it must also provide usable truth when context is strong enough. In full-scope ranking imports, power order is valid rank context because Last War ranking screens are sorted by power.

## v0.9.5.92 Lesson – Preserve Evidence and Repair Output Separately

A repaired final rank must not erase OCR evidence. `visible_rank` and `ocr_rank` remain the observed/raw surfaces. `rank`, `operational_rank`, and `final_rank` may be repaired or inferred when the context is strong. This separation keeps Sentinel explainable.

## v0.9.5.91 Lesson – Screenshot First, Export Second

The screenshot is the only ground truth. Export, console summary, Review Dashboard and Evidence Pack are Sentinel outputs and must be validated against screenshots, never the other way around.

## v0.9.5.91 Lesson – Rank Context Is a Container

Rows from a screenshot window must not be promoted into unrelated rank slots just because their power value sorts high. Rank context is a data-quality boundary.

## v0.9.5.91 Lesson – High Ranks Are Not Invalid by Themselves

Rank 203 or 300 can be legitimate in a future THP Top-300 context. The error is not the number alone; the error is a rank that violates the recognized ranking mode and screenshot context. The correct failure family is `rank_scope_violation`, not a blanket high-rank ban.

## v0.9.5.91 Lesson – Full Ground Truth Is Expensive

Server 551 THP remains the current gold-standard Ground Truth set. Full manual Ground Truth for every server would be too expensive; use targeted mini-GT only for new P0 failures.
